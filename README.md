# master-project-ascon
Benchmarking ASCON (NIST LWC) against AES and Speck on constrained IoT hardware (ARM, RISC-V, MSP430). This research evaluates memory usage, energy-per-bit, and security tradeoffs including, side-channel vulnerability and post-quantum readiness (ASCON-80pq), to bridge the gap between theoretical efficiency and physical deployment.
